About

This is a toolset designed to render a given model within a CAVE Virtual Reality suite or on a device capable of displaying WebGL graphics.

How to configure the toolset:

  1. Model - Place the model(s) that you wish to use within the /models folder of the toolset structure. The toolset will not be able to locate your models otherwise.
  2. Texture - If your model uses a texture, place it in the /models/textures folder of the toolset structure. The toolset will not be able to locate your texture otherwise.
  3. Viewing mode - If you're using the toolset on a monitor or another flat device, set the viewing mode to 'Flat screen'. If you're using the toolset within a CAVE suite, set it to 'CAVE'.
  4. Orientation - The orientation of each seperate screen within your environment.
  5. Screen rotation - This is the angle at which each of the screens are rotated from the centre screen.
  6. Field of view - This is amount of the model that is viewable from each screen.
  7. Lighting - Whether or not there needs to be lighting within the scene.
